# 📊 HỆ THỐNG QUẢN LÝ NHÂN VIÊN (HRM)

## 🧾 Giới thiệu

Ứng dụng Quản lý nhân viên (HRM) được xây dựng bằng C# với giao diện GUI
thân thiện, hỗ trợ quản lý thông tin nhân viên, chấm công, tính lương và
báo cáo.

------------------------------------------------------------------------

## 🚀 Tính năng chính

### 👨‍💼 Quản lý nhân viên

-   Thêm / sửa / xóa nhân viên\
-   Tìm kiếm và hiển thị danh sách

### ⏱️ Chấm công

-   Ghi nhận giờ vào / giờ ra\
-   Trạng thái: Đang làm, Nghỉ, Hoàn tất

### 📅 Ngày công

-   Thống kê ngày làm việc\
-   Quản lý nghỉ phép

### 💰 Tính lương

-   Dựa trên ngày công và tăng ca\
-   Xuất báo cáo

### 📈 Báo cáo

-   Xuất CSV\
-   Thống kê tổng quan

### ⚙️ Cài đặt

-   Dark / Light mode

------------------------------------------------------------------------

## 🛠️ Công nghệ

-   C#
-   WinForms / WPF
-   JSON / SQL Server

------------------------------------------------------------------------

## 📂 Cấu trúc

HRM/ │── Models/ │── Services/ │── Forms/ │── Data/ │── Program.cs

------------------------------------------------------------------------

-   Mở bằng Visual Studio\
